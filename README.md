# 🚗 Premium Motors — Sistema Completo de Concessionária

Projeto Full Stack desenvolvido com Next.js focado em uma experiência moderna para concessionárias de veículos, contendo painel administrativo, gerenciamento de carros, sistema de interesse de clientes, área de desejos personalizados, upload de imagens e gerenciamento de mídia.

---

# 📸 Preview do Projeto

## Home
- Carousel de banners
- Lojas oficiais
- Listagem dinâmica de veículos
- Cards responsivos

## Área de Detalhes
- Galeria de imagens
- Informações completas do veículo
- Formulário de interesse

## Painel Administrativo
- Cadastro de veículos
- Edição e exclusão
- Gerenciamento de banners
- Gerenciamento de lojas oficiais
- Controle de interesses dos clientes

---

# ✨ Funcionalidades

## 🚘 Sistema de Veículos
- Cadastro completo de carros
- Upload de múltiplas imagens
- Exibição dinâmica dos veículos
- Página individual de detalhes
- Valores formatados em Real (BRL)

---

## 🧑‍💼 Painel Administrativo
- Dashboard administrativo
- Gerenciamento de carros
- Edição de informações
- Exclusão de veículos
- Controle centralizado do sistema

---

## ❤️ Sistema de Interesse
Clientes podem:
- Demonstrar interesse em veículos
- Enviar:
  - Nome
  - Email
  - Telefone
  - Mensagem personalizada

O administrador consegue visualizar todos os interesses no dashboard.

---

## 🔍 Sistema "Encontre Meu Carro"
Caso o cliente não encontre o veículo desejado:

Ele pode enviar:
- Nome
- Contato
- Modelo desejado
- Marca
- Ano
- Cor
- Preferências

Tudo é salvo no banco de dados para acompanhamento.

---

## 🖼️ Sistema de Mídia
Painel dedicado para:
- Cadastro de banners da home
- Gerenciamento das lojas oficiais
- Upload de imagens via Cloudinary

---

## ☁️ Upload de Imagens
Integração com Cloudinary para:
- Upload de banners
- Upload de imagens das lojas
- Armazenamento em nuvem
- Melhor performance e escalabilidade
